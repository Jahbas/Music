# Liquid Music
Best fully open source music player.
React + Vite + TypeScript. Music library and player; tracks and playlists stored in IndexedDB.

## Setup

```bash
npm install
```

## Scripts

- `npm run dev` — dev server
- `npm run build` — production build
- `npm run preview` — serve the build locally

## Stack

- React 18, React Router, Zustand
- Vite 5, TypeScript
- idb, music-metadata-browser (for file import and metadata)
